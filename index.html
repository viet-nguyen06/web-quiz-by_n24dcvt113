Ôn tập trắc nghiệm

ZIP cần có questions.yaml (hoặc questions.yml) ở gốc ZIP và các file ảnh (nếu có). Trong YAML, ảnh dùng cú pháp [img=đường_dẫn_trong_zip].

Tải tài liệu mẫu: Click để tải file Demo.zip

Cách tạo file .zip để ôn tập:

  • Bước 1: Tạo file nội dung

    Tạo một file văn bản có tên là questions.yaml. Cấu trúc nội dung phải tuân thủ định dạng trong file demo.

  • Bước 2: Soạn thảo câu hỏi
    • Tăng id theo số thứ tự câu hỏi (1, 2, 3...).
    • Sử dụng ký hiệu ## cho các lựa chọn sai.
    • Sử dụng ký hiệu #$ cho lựa chọn đúng duy nhất.
    • Hỗ trợ công thức Toán học bằng **Latex**.
  • Bước 3: Chèn hình ảnh (nếu có)

    Sử dụng cú pháp [img=ten_hinh.png] trong nội dung câu hỏi. Lưu ý các file hình ảnh phải nằm **cùng thư mục** với file questions.yaml.

  • Bước 4: Nén file

    Chọn file questions.yaml và các hình ảnh liên quan, sau đó **nén lại (Zip)** thành một file duy nhất để tải lên hệ thống.
